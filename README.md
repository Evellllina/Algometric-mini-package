## Лабораторная работа №3
## Алгоритмический мини-пакет (структуры данных и сортировки)
## Цель:
Освоить принципы работы и реализовать основные алгоритмы сортировок (пузырьковую, быструю, подсчетом, поразрядную, блочную и пирамидальную), проанализировать базовые структуры данных(стек и очередь).
## Генераторы:
1) rand_int_array – массив случайных целых чисел.
2) nearly_sorted – почти отсортированный массив (с перестановками).
3) many_duplicates – массив с большим количеством дубликатов.
4) reverse_sorted – массив, отсортированный в обратном порядке.
5) rand_float_array – массив случайных чисел с плавающей точкой (float).
## Бенчмаркинг:
Функция benchmark_sorts измеряет время выполнения каждой сортировки
## Структур данных:
1) Стек
2) Очередь
## Алгоритмы сортировок:
1) Пузырьковая
2) Быстрая
3) Подсчетом
4) Поразрядная
5) Блочная
6) Пирамидальная
## Особенности:
1) Command Line Interface — интерфейс командной строки (принимает входные данные, выполняет ссответствующие действия, выводит результат в консоль)
2) Валидация входных параметров
